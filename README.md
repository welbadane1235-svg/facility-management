- حساب الباحث لا يظهر له نشر وظيفة.
- قسم النشر يظهر فقط نموذج ملف باحث عن عمل.
- إرفاق صورة تعريفية.
- إرفاق CV PDF.
- صفحة بيانات شخصية مستقلة.
- صفحة إشعارات مستقلة.
- إشعار عند مشاهدة CV.
- زر تمت المشاهدة للإشعارات.
