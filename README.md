# HireX Candidate Clean V3

## الجديد
- رجوع قائمة الوظائف في الصفحة العامة.
- إضافة أقسام الوظائف.
- إضافة زر "الوظائف" داخل القائمة الجانبية.
- صفحة الوظائف مستقلة داخل لوحة الباحث.
- عرض الوظائف حسب القسم.
- زر تقديم على الوظيفة للباحث.

## التحديث
1. شغل `supabase-setup.sql` في Supabase.
2. ارفع `index.html` بدل القديم.
3. حدث الصفحة.
